Consigna
- Crear una lista de tareas (to-do-list) en HTML que incluya un campo de texto de entrada y un botón de "Agregar tarea"
- Cuando el usuario introduzca una tarea y presione el botón, ésta deberá añadirse a una lista de tareas que en un principio deberá estar vacía.
- Cada tarea en la lista deberá estar acompañada por un botón "Eliminar" a su lado.
- Al hacer clic en dicho botón, la tarea correspondiente deberá ser eliminada de la lista.

Pistas y sugerencias
- Utilizar JavaScript como lenguaje de programación para el desarrollo.
- Investigar sobre diferentes métodos del DOM que puedan ser utilizados para este desarrollo.
- Intentar mejorar un poco el diseño poniendo en práctica conceptos básicos de CSS.
